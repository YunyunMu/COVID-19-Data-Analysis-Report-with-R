# Plot the Covid-19 data with R for top 10 countries
This is a beginner level project for R plotting to resolve the real-world challenges.  By applying data wrangling techniques such as grouping and summarizing, it shows which countries have the highest rates of positive COVID-19 tests relative to their testing numbers. 

Language: R
Packages: dplyr
          readr
          tibble
          ggplot2

Dataset: test_worldwide.csv

Outcomes: 
Creating and manipulating different R data structures (vectors, matrices, lists) and plotted the top 10 countries by thier positivity rate.
